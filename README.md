# Krythor

Local-first AI command platform with intelligent model routing, memory, and agent execution.

## 🚀 What is Krythor?
Krythor is a local-first AI system that lets you run agents, manage memory, and route tasks across multiple models — all from a single control interface.

## ✨ Features
- Local-first AI execution
- Multi-model routing (OpenAI, Anthropic, Ollama, GGUF)
- Agent system with fallback logic
- Persistent memory engine
- Transparent execution (see why models were chosen)
