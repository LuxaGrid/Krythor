# ERROR PLAYBOOK

A running log of recurring bug patterns and their fixes.
The goal is: **fix it once, never chase it again**.

## Entry Template
## <Short title>
**Date:** YYYY-MM-DD
**Symptom:** What you saw (exact error)
**Root cause:** Why it happened (not the cascade)
**Fix:** What changed (files + key lines)
**Verify:** Commands to confirm
**Prevention:** Test / guard / lint / doc change
